# Glossary

A brief dictionary of words related to **Information Retrieval**.

## Core Concepts

| Term                       | Definition                                                                                                         |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| Information Retrieval (IR) | The task of finding material (usually documents) that satisfies an information need from within a large collection |
| Document                   | Any unit of text being indexed and retrieved - a webpage, paragraph, paper, etc.                                   |
| Corpus                     | The full collection of documents being searched                                                                    |
| Query                      | The user's expression of an information need                                                                       |
| Relevance                  | How well a document satisfies a query - can be binary (relevant/not) or graded                                     |
| Information Need           | The underlying thing the user actually wants to know, which may differ from the query they type                    |
